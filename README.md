# Todo App

A simple **To-Do List application** built with **React** and planned to integrate with the **MERN stack**.  

Users can add, delete, and mark tasks as complete or incomplete. This project is a learning exercise in **frontend development**, with plans to add backend integration for persistent storage.

---

## Features

- Add new tasks  
- Delete existing tasks  
- Mark tasks as complete or incomplete  
- Responsive and simple UI

---

## Tech Stack

- **Frontend:** React  
- **Backend (planned):** Node.js, Express  
- **Database (planned):** MongoDB  
- **Version Control:** Git & GitHub
